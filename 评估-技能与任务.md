# Trading 212 插件开发 — 任务评估与技能映射

## 完成任务所需项（评估）

| # | 所需项 | 说明 | 使用的 Skill |
|---|--------|------|--------------|
| 1 | 创建「如何做 Trading 212 扩展」的 Skill | 无现成 Chrome 扩展 / Trading 212 skill，需先有指导文档供执行方遵循 | **create-skill**：按 Cursor 规范创建新 skill |
| 2 | 创建负责执行的 Subagent | 需要专用 agent 按 skill + PLAN 执行开发，而非主对话直接写代码 | **create-subagent**：创建 trading212-extension-builder 子 agent |
| 3 | 执行开发（扩展 + 可选代理 + README） | 实现 manifest、popup、options、api 层、UI、代理、文档 | **新建的 skill + subagent**：通过 mcp_task 调用 subagent，subagent 遵循 skill 与 PLAN 执行 |

## 执行顺序（全部基于 Skill）

1. **使用 create-skill**  
   - 创建项目级 skill：`.cursor/skills/trading212-extension/`  
   - 内容：实现步骤、文件结构、API 用法、UI 要求（来自 PLAN.md、可行性说明.md）

2. **使用 create-subagent**  
   - 创建项目级 subagent：`.cursor/agents/trading212-extension-builder.md`  
   - 描述：在收到“实现 Trading 212 扩展”任务时，读取 PLAN、可行性说明与 trading212-extension skill，并完整实现扩展、可选代理与 README

3. **通过 skill 驱动执行**  
   - 调用 mcp_task（generalPurpose），提示词中引用上述 subagent 说明 + skill 路径 + PLAN/可行性说明  
   - 由 subagent/agent 按 skill 与计划完成所有代码与文档，主对话不直接写实现代码

## 技能来源确认

- **create-skill**：`~/.cursor/skills-cursor/create-skill/SKILL.md`
- **create-subagent**：`~/.cursor/skills-cursor/create-subagent/SKILL.md`
- **trading212-extension**（新建）：`TradingAgent/.cursor/skills/trading212-extension/SKILL.md`
- **trading212-extension-builder**（新建）：`TradingAgent/.cursor/agents/trading212-extension-builder.md`

---

## 执行说明（因 mcp_task 超时）

原计划通过 mcp_task 调用 generalPurpose 子 agent 按 skill 执行开发；该调用超时。后续由主 agent **严格按 `.cursor/skills/trading212-extension/SKILL.md` 检查清单**完成实现，以保证「所有执行基于 skill」。
