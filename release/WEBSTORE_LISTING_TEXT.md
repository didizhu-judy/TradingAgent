## Extension Name
T212 Agent

## Short description (<=132 chars)
Trading 212 account overview, side panel dashboard, and watchlist with searchable symbols and auto-updating prices.

## Detailed description
T212 Agent helps you track your Trading 212 account in a lightweight browser panel.

Key features:
- Account overview: total value, available cash, investments value, realized/unrealized P&L.
- Positions view: sortable holdings by value, P&L, or name.
- Open in side panel: pin your dashboard in Chrome side panel for continuous monitoring.
- Watchlist page: search symbols and add to watchlist if not already included.
- Auto refresh: periodically refresh account and watchlist quotes.
- AI analysis panel: generate market analysis prompts and open external analysis tools.

Data behavior:
- Credentials and settings are stored locally in your browser storage.
- Data is fetched from configured APIs only for account and quote display.
- No user browsing history is collected or sold.

Notes:
- You need Trading 212 API credentials to load account data.
- Optional proxy can be enabled for environments with CORS limitations.
